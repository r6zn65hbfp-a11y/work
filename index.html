关于本站

这是一个单文件模板（仅此 index.html），方便你直接发布。它包含：响应式网格、灯箱、可复用的泪滴 SVG、以及温和的玻璃拟态风格。你可以把它改成作品集、专题页面、或“寻宝”图景式展示。

  • 替换 Logo 与标题：在导航处把 SVG 与站名改成你自己的。
  • 添加图片：把文件放到 /assets 文件夹，并替换每个 figure.tile 的 data-src 与 img src。
  • 自定义主色：在 :root 里改 --accent/--accent-2。

如何上线（公开给所有人看）

  1. 在电脑上创建文件夹（如 my-site），把本文件保存为 index.html。另建 assets 子文件夹放图片。
  2. 本地预览：双击打开或在 VS Code 中使用 Live Server 预览。
  3. 免费部署方式 A · Netlify：将整个文件夹拖到 Netlify Drop（无需登录亦可快速测试）。
  4. 免费部署方式 B · GitHub Pages：
    • 注册 GitHub → 新建仓库 → 上传 index.html 与 assets。
    • 在仓库 Settings → Pages → Source 选择 main 分支，保存后会生成公开网址。
  5. 免费部署方式 C · Vercel：把仓库导入 Vercel，一键部署，获得公开域名。

想进一步设计？用 Figma 画一个版式草图（手机/电脑两版），再把样式落实到这份模板即可。
